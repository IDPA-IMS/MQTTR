How It Works

This web controller connects to an MQTT broker via WebSocket using mqtt.js. When you press a key, it sends a message to the corresponding MQTT topic. Messages you send appear in the log as [OUT], and messages from other clients appear as [IN].

Notes:
  • The log auto-scrolls as new messages arrive.
  • MQTT topics are move/# by default;
